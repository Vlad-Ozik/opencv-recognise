Подробное описание

Принимает данные (изображения) с клиента и находит на них контуры, которые похожи на контуры заданного изображения

Функция для нахождения контуров переданных изображений

Аргументы
imgизображение, которое прислали на сервер
i_cropизображение, контур которого будет искатся
Возвращает
cnts, cnts_c - контуры изображений img и i_crop соотвественно

Для нахождения контуров производятся действия:

  • сделать img черно-белой, i_crop уже ч/б
  • размыть изображение
  • обнаружить края объектов на изображении
  • произвести морфологическую трансформацию
  • найти контур

Функция для поиска похожих контуров

Аргументы
imgизображение, которое прислали на сервер
i_cropизображение, контур которого будет искатся
spanвсё в диапазоне от -span до span является похожим контуром
Возвращает
img конечное изображение с выделеным похожим контуром, если он есть

Для нахождения похожих контуров производятся действия:

  • найти контуры изображений с помощью функции find_contours
  • в цикле найти Ху моменты контуров изображения i_crop и занести их в массив
  • в цикле в цикле найти Ху моменты контуров изображения img и сравнить их c Ху моментами контуров изображения i_crop, если разница между их ними находится между -span и span, то контур наноситься на img

По умолчанию параметр span=0.028. Для нахождения похожих контуров с веб камеры плохого качества это значения является наилучшим (было полчено эксперементальным путём). Если изображения хорошего качества, параметр span можно уменьшить, чтобы уменьшить риск нахождения не схожих контуров.

  i_crop,
  comp_factor = 0.1 
)

Альтернативная функция для поиска похожих контуров

Аргументы
imgизображение, которое прислали на сервер
i_cropизображение, контур которого будет искатся
comp_factorмаксимальный показатель схожести контуров
Возвращает
img конечное изображение с выделеным похожим контуром, если он есть

Для нахождения похожих контуров производятся действия:

  • найти контуры изображений с помощью функции find_contours
  • в цикле сравниваются сами контуры с помощью функции matchShapes, которая возвращяет показатель сходства контуров

По умолчанию параметр comp_factor=0.1. Для нахождения похожих контуров с веб камеры плохого качества это значения является наилучшим (было полчено эксперементальным путём). Если изображения хорошего качества, параметр span можно уменьшить, чтобы уменьшить риск нахождения не схожих контуров.
